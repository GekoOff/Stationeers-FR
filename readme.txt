Installation de la Traduction Française pour Stationeers

Mise à jour / Version : 0.2.4767.21868

Une erreur ? Venez sur discord : https://discord.gg/dSxvNKehwy



Étapes d'Installation :

1. Téléchargement des Fichiers :
   - Cliquez sur le lien suivant pour télécharger les fichiers de traduction : https://www.mediafire.com/folder/6ijuroc0amen1/Stationeers+FR.
   - Extrayez le contenu du fichier ZIP dans un emplacement de votre choix sur votre ordinateur.

2. Emplacement des Fichiers dans le Jeu :
   - Trouvez le dossier d'installation de Stationeers sur votre ordinateur. Cela peut varier en fonction de la plateforme.
     - **Steam :** `C:\Program Files (x86)\Steam\steamapps\common\Stationeers`
     - **Autre :** Recherchez le dossier d'installation de Stationeers sur votre système.

3. Copie des Fichiers de Traduction :
   - Ouvrez le dossier "Stationeers" et localisez le dossier "Lang" à l'intérieur (C:\Program Files (x86)\Steam\steamapps\common\Stationeers\rocketstation_Data\StreamingAssets\Language.
   - Copiez les fichiers de traduction téléchargés (généralement des fichiers .xml) dans le dossier "Lang".
   - Supprimer les anciens fichiers de base traduits.

4. Lancement du Jeu avec la Traduction :
   - Lancez Stationeers comme d'habitude.
   - Allez dans les paramètres du jeu et sélectionnez "French" comme langue préférée.

5. Vérification et Retours :
   - Explorez le jeu pour vous assurer que la traduction fonctionne correctement.
   - Si vous repérez des erreurs ou avez des suggestions, veuillez nous les signaler par message privé sur Discord : https://discord.gg/dSxvNKehwy.

## Remarques Importantes :

- Assurez-vous de sauvegarder les fichiers originaux du jeu avant d'ajouter la traduction, au cas où vous voudriez revenir à la version originale.
- Cette traduction est en constante amélioration. Vos retours et suggestions sont précieux pour nous aider à fournir la meilleure expérience possible.

Merci de soutenir la communauté de traduction Stationeers_FR !
